# Kwacha Forex App

A **modular Streamlit app** for visualizing, analyzing, and forecasting Zambian Kwacha (ZMW) FX rates using ML models and walk-forward backtesting.

---

## Features

- **Interactive visualizations:** Mid rates, moving averages, volatility  
- **ML Forecasts:** Linear Regression, Random Forest, Decision Tree, XGBoost  
- **Walk-forward backtesting:** Rolling evaluation to avoid data leakage  
- **Modular design:** Home, Forecasts, Walk-Forward Backtest, About pages  
- **Professional UI:** Full-width layout, sidebar navigation with icons, deep-linking
